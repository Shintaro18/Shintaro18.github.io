What's Mr. Behavior?



Mr. Behavior is a free & simple behavior analysis software package that runs on Windows computers.
The included software is as followed,

Tracker
A tracking software allows ROIs to be set arbitrarily. It can be used for multiple behavioral analyses such as open field, three chamber social interaction test, light/dark choice test, elevated cross maze etc.


FreezeAnalyzer
Freezing analysis software for fear conditioning etc.


BinTimer
Timer to measure the time spent pressing keys on the keyboard for human count analysis.


AviRecorder
Recording software. It can record from multiple USB cameras simultaneously.
