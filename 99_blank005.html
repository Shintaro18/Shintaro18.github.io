About Option Parts



The Operant House is designed to allow configuration changes depending on the user's needs.
This section explains which options you should choose.



Your current setup would look like above. This is the minimal configuration optimized for space and cost. The following construction path will branch depending on your intended use. Please choose the option that best fits your needs:

A Use the current setup (This applies to most users)
--> Proceed to "IR Camera Detection"

B You want to run tasks involving a lever
--> Proceed to "Lever Module" (Can be combined with options A and C)

C You want to run tasks with multiple vertical rows of panels (e.g., Pattern Separation with the panels arranged in 5 columns and 2 rows)
--> Proceed to "IR Sensor Bar"

The current setup does not include a secondory monitor for operation. If you put a mouse with this setup, the device can only be operated via another PC because we cannot see the touch-monitor. If you’d like to operate the device without remote desktop, add the secondory monitor by following "Additional Operation Monitor" after completiing the setup steps for options A, B, or C.
