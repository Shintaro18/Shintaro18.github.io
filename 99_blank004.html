Production of the OperantHouse


It's time to start building the OperantHouse. The model data (STL file) can be found under "Preparation (Download)" page so download it for the following steps. Once you download it, find STL files and open it with slicer software such as Cura to convert it into G-code data which 3D printer can recognize. Printing conditions and part details can be found in the "Part Printing" page.
